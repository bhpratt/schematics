# schematics-vpcgen2-iks

Creates a single-zone VPC Gen2 IBM Cloud Kubernetes Service cluster. To be used with Terraform or IBM Cloud Schematics. Allows for a quick build and teardown of a Kubernetes and VPC environment.

source: [IBM Cloud Terraform provider](https://github.com/IBM-Cloud/terraform-provider-ibm/tree/master/examples/ibm-cluster/vpc-gen2-cluster)
